def quick_sort(arr): if len(arr) <= 1: return arr pivot = arr[len(arr) // 2] left = [x for x in arr if x < pivot] middle = [x for x in arr if x == pivot] right = [x for x in arr if x > pivot] return quick_sort(left) + middle + quick_sort(right)

🎨 字体特点分析

JetBrains Mono 特点：

  • ✅ 专为代码设计，字符间距均匀
  • ✅ 清晰的字符区分（0/O, 1/l, I/i）
  • ✅ 优秀的可读性和专业外观
  • ✅ 支持连字功能（如果可用）

字体优先级：

  1. JetBrains Mono - 首选，最专业的代码字体
  2. Fira Code - 备选，支持连字
  3. SF Mono - macOS 系统字体
  4. Monaco - 经典字体
  5. Cascadia Code - 微软开源字体
  6. Consolas - Windows 经典字体
  7. Monospace - 系统默认等宽字体
